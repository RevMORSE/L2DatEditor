Для ру выбираем хроники ru_helios

Если dat файлы оригинальные то крипт v413_original
Если нет то v413_encdec

После фулл распаковки нужно поправить корейские ошибки путем замены во всех файлах ( либо поиском )
[]] -> [None]
[skill_end] -> [SpAtk01]
они так же видны в логе при распаковке.


Не рекомендуется использовать в текстовых полях [ и ].

при DAT_REPLACEMENT_NAMES=true [config_debug.ini]
Нельзя менять датки по одной если в них содержится MAP_INT ( только полная перепаковка ).

Если файл не собирается или не разбирается включаем конфиг DAT_DEBUG_MSG и DAT_DEBUG_POS [config_debug.ini].
